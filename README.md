# Line Follower

![Line Follower](Media/Final.jpeg)

A simple autonomous line follower robot built using an Arduino UNO, infrared sensors, DC motors and an L298N motor driver.

The robot detects a dark line on the ground and adjusts the motor speeds to stay on track using basic feedback control.

Watch demo: https://youtu.be/aXs3XGMi7pI

## What I learned

- Reading infrared sensor values
- Controlling DC motors through an L298N motor driver
- Basic feedback loop logic
- Designing and CAD modeling a robot chassis
- Powering multiple components from a single battery

## Challenges

- Designing the chassis and electronics holders in CAD
- Routing and mounting the electronics cleanly
- Powering the motors and control board from one battery
- Tuning the robot so it follows the line reliably

## Limitations

- The motors are relatively slow and could be upgraded
- The control logic is simple and could be improved with PID
- The robot currently relies only on basic infrared sensor readings
